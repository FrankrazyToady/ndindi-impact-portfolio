# Ndindi Nyoro – Digital Portfolio

## Overview
This is a **digital portfolio for Samson Ndindi Nyoro**, Member of Parliament for Kiharu Constituency, entrepreneur, and economist.  
The portfolio highlights his **leadership journey, achievements, constituency projects, quotes, and personal values** in a professional and interactive format.  

This project was built as a **tech showcase** to demonstrate web development skills and the ability to present impactful leadership digitally.

---

## Features
- **Hero Section**: Name, role, headshot, and social media links  
- **Profile Overview**: Short summary of leadership and contributions  
- **Education**: Academic background in card layout  
- **Professional & Business Achievements**: Highlights business and mentorship initiatives  
- **Political Leadership**: Parliamentary roles, contributions, and key achievements  
- **Constituency Projects**: Tabbed interface for Education, Youth Empowerment, and Infrastructure projects  
- **Quotes Carousel**: Interactive slider with notable quotes  
- **Stats Counters**: Animated counters for measurable impact (optional)  
- **Contact Section**: Constituency office, email/social handles, and website  

---

## Tech Stack
- **HTML5** – structure of the website  
- **CSS3** – styling, responsive design, hover effects  
- **JavaScript** – interactivity (quotes carousel, tabs, counters)  
- **Assets** – images/icons stored locally in `/assets/`  
- **Hosting** – can be deployed via Netlify, Vercel, or GitHub Pages

## How to Run
1. Clone or download this repository.  
2. Open `index.html` in any web browser.  
3. Ensure `css/`, `js/`, and `assets/` folders are intact for full functionality.  
4. Optional: Deploy online using Netlify, Vercel, or GitHub Pages.
